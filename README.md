## Локальный запуск

Чтобы запустить проект локально, склонируйте его:

`git clone https://github.com/00unnmd/haulmont-qt.git`

Перейдите в директорию склонированного проекта: 

`cd haulmont-qt/`

Установите зависимости: 

`yarn install`

И запустите :) 

`yarn start`
